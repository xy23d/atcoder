# AGENTS.md

このリポジトリで AI エージェント（Codex / ChatGPT など）が作業・レビューするときのガイドラインです

## プロジェクト概要

- 用途: atcoderの回答・テンプレート管理
- 言語: Go

## ディレクトリ構成（ざっくり）

- `answers/` … 回答
- `templates/` … 再利用・よく使うコード
- `wips/` … 作業途中 (削除予定)

## コーディングスタイル

- Go: go fmt に準ずる

## テスト / 実行

- 特になし

## AI へのお願い

- コメント・説明は **日本語** で
- レビュー時:
  - プレフィックスをつけること `[must]・[should]・[nits]・[imo]`
  - 重大なバグになりそうな点ははっきり指摘してOK
